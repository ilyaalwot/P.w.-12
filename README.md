# Практична робота №12
## Робота з потоками введення/виведення. Фільтрація файлів журналу

У даній практичній роботі реалізовано програму для фільтрації великих файлів системного журналу (*.log).  
Програма зчитує лог-файл рядок за рядком, знаходить записи з вибраним рівнем логування та записує їх в окремий файл.

## Реалізовано:
- перелічення `LogLevel` для рівнів логування;
- інтерфейс `LogFilter` з методом `filter(...)`;
- клас `SevereLogFilter`, який виконує фільтрацію рядків за рівнем логування;
- потокове зчитування файлу без завантаження всього в пам’ять;
- використання `try-with-resources`;
- ланцюг введення: `FileReader -> BufferedReader -> Scanner`;
- ланцюг виведення: `FileWriter -> BufferedWriter -> PrintWriter`;
- генерацію тестового лог-файлу;
- консольне меню для демонстрації роботи програми.

## Рівні логування:
- CRITICAL
- ERROR
- WARNING
- INFO
- DEBUG

## Використані технології
- Java
- Enum
- Interface
- FileReader / BufferedReader / Scanner
- FileWriter / BufferedWriter / PrintWriter
- try-with-resources
